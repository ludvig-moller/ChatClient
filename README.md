# ChatClient

När programmet startar börjar användaren med att skriva in sitt användarnamn. Där efter kommer användaren in i ett standard läge utanför alla chattrum. Här kan kommandon användas, till exampel `/join general` för att gå in i rummet "general".

## Kommandon
`/help` - Visar en lista med alla tillgängliga kommandon och vad det används för.

`/quit` - Avslutar applikationen och går ut ur eventuellt rum.

`/join <rum>` - Går med i det specificerade rummet.

`/leave` - Lämnar det nuvarnade rummet.

`/rooms` - Visar en lista på alla tillgängliga rum.

`/dm <mottagare> <meddelande>` - Skickar ett privat meddelande till mottagaren.

`/clear` - Rensar konsolen på meddelanden.

`/ignore <användarnamn>` - Ignorerar en användares kommande meddelanden.

`/unignore <användarnamn>` - Slutar ignorera användaren.

`/ignored` - Visar en lista på de användare som ignoreras.
